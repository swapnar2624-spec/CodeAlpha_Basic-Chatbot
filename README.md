Basic Chatbot

## Project Description
This is a simple rule-based chatbot developed using Python as part of the CodeAlpha Internship.

The chatbot responds to predefined user inputs and provides appropriate replies.

## Features
- Responds to greetings like "hello"
- Answers simple questions
- Handles unknown inputs
- Exits when the user types "bye"

## Technologies Used
- Python

## Concepts Used
- Functions
- While Loop
- If-Elif-Else Statements
- User Input Handling
- String Methods
